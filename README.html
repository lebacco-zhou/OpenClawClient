🚀 OpenClaw Client

安全加密的 Windows 桌面聊天客户端

📦 功能特性

端到端加密通信（AES-256-GCM）
登录配置管理（服务器地址、Gateway Token）
文件拖拽发送
剪贴板粘贴图片/文件
自定义文件下载路径
自动日期子文件夹分类
绿色版 + 安装包版双版本
连接状态指示器
消息发送状态跟踪
文件下载进度显示
图片全屏查看器（缩放、拖拽、保存）
消息搜索功能
自动重连机制（最多 5 次，指数退避）

🛠️ 技术栈

.NET 8.0 WPF MaterialDesignInXaml AES-256-GCM 加密 DPAPI 本地数据保护 WebSocket HttpClient

📁 项目结构

OpenClawClient/ ├── src/ │ ├── OpenClawClient.Core/ # 核心业务逻辑 │ │ ├── Models/ # 数据模型 │ │ └── Services/ # 服务层（加密、网络、配置） │ ├── OpenClawClient.UI/ # WPF 界面 │ │ ├── Views/ # 登录、聊天、图片查看器等 │ │ └── Converters/ # 值转换器 │ └── OpenClawClient.Desktop/ # 桌面应用入口 ├── tests/ # 单元测试 ├── scripts/ # 发布脚本 │ ├── publish-portable.ps1 # 绿色版发布 │ └── publish-setup.ps1 # 安装包发布 ├── installer/ # Inno Setup 脚本 └── dist/ # 输出文件

🚀 快速开始

环境要求

  • Windows 10/11 (64 位)
  • .NET 8.0 SDK
  • Visual Studio 2022 (可选)
  • Inno Setup 6 (安装包编译，可选)

本地运行

cd OpenClawClient
dotnet run --project src\OpenClawClient.Desktop

发布绿色版

.\scripts\publish-portable.ps1

输出：dist/OpenClawClient_Portable.zip (~70MB)

发布安装包版

.\scripts\publish-setup.ps1

输出：dist/OpenClawClient_Setup.exe (~75MB)

🔒 安全说明

  • 敏感字段使用 DPAPI 加密存储
  • 消息内容使用 AES-256-GCM 端到端加密
  • 文件传输使用 HTTPS 加密通道
  • 不存储明文密码或令牌

📝 配置文件

配置路径

  • 绿色版: ./config/appsettings.json
  • 安装包版: %APPDATA%\OpenClawClient\appsettings.json

配置示例

{ "serverUrl": "https://www.lebacco.cn:8443", "gatewayToken": "<31 位十六进制令牌>", "aesKey": "<AES-256 密钥>", "downloadPath": "D:\\MyFiles\\OpenClaw", "autoSubfolder": true, "rememberLogin": true }
