## 🔒 安全特性

| 特性 | 实现方式 |
|------|----------|
| **消息加密** | AES-256-GCM 端到端加密 |
| **密钥长度** | 256 位（32 字节） |
| **IV 长度** | 12 字节（GCM 推荐） |
| **认证标签** | 16 字节 |
| **本地存储** | DPAPI（CurrentUser 范围） |
| **传输加密** | HTTPS/WSS |
| **敏感字段** | Gateway Token、AES 密钥加密存储 |

## 📝 配置文件

### 配置位置
- **绿色版**: `./config/appsettings.json`
- **安装包**: `%APPDATA%\OpenClawClient\appsettings.json`

### 配置格式
```json
{
  "serverUrl": "https://www.lebacco.cn:8443",
  "gatewayToken": "<encrypted>",
  "aesKey": "<encrypted>",
  "downloadPath": "D:\\MyFiles\\OpenClaw",
  "autoSubfolder": true,
  "rememberLogin": true,
  "windowPosition": {
    "left": 100,
    "top": 100,
    "width": 1200,
    "height": 800
  }
}
```
