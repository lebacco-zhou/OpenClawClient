# OpenClaw Client 开发指南

## 🎯 项目概述

OpenClaw Client 是一个安全加密的 Windows 桌面聊天客户端，提供与 OpenClaw Gateway 的可靠通信。

## 📋 开发环境配置

### 必需软件

1. **.NET 8.0 SDK**
   - 下载地址：https://dotnet.microsoft.com/download/dotnet/8.0
   - 验证安装：`dotnet --version`

2. **Visual Studio 2022** (推荐)
   - 社区版免费：https://visualstudio.microsoft.com/
   - 安装时选择".NET 桌面开发"工作负载

3. **Inno Setup 6** (安装包编译，可选)
   - 下载地址：https://jrsoftware.org/isdl.php
   - 用于编译 `Setup.exe` 安装包

### 可选工具

- **Git** - 版本控制
- **VS Code** - 轻量级编辑器
- **ReSharper** - 代码分析和重构工具

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone <repository-url>
cd OpenClawClient
```

### 2. 还原依赖

```bash
dotnet restore
```

### 3. 运行调试

```bash
dotnet run --project src/OpenClawClient.Desktop
```

或在 Visual Studio 中：
- 打开 `OpenClawClient.sln`
- 设置 `OpenClawClient.Desktop` 为启动项目
- 按 F5 启动调试

## 📦 发布构建

### 绿色版（Portable）

```powershell
# PowerShell
.\scripts\publish-portable.ps1
```

输出：`dist/OpenClawClient_Portable.zip`

**特点：**
- 解压即用
- 配置文件在 `./config/appsettings.json`
- 适合 U 盘携带

### 安装包版（Setup）

```powershell
# PowerShell
.\scripts\publish-setup.ps1
```

输出：`dist/OpenClawClient_Setup.exe`

**特点：**
- 标准 Windows 安装流程
- 配置文件在 `%APPDATA%\OpenClawClient\`
- 自动创建快捷方式
- 支持卸载

## 🔒 加密说明

### 消息加密

- **算法**: AES-256-GCM
- **密钥长度**: 256 位
- **IV 长度**: 12 字节（GCM 推荐）
- **认证标签**: 16 字节
- **密钥交换**: 手动配置或 RSA 加密传输

### 本地数据保护

- **配置存储**: DPAPI（CurrentUser 范围）
- **敏感字段**: Gateway Token、AES 密钥
- **安全级别**: 仅当前用户可解密

## 🐛 常见问题

### Q: 连接失败
**A:** 检查：
1. 服务器地址是否正确
2. Gateway Token 是否有效
3. 防火墙是否阻止 8443 端口
4. WebSocket 代理配置

### Q: 解密失败
**A:** 检查：
1. AES 密钥是否正确配置
2. 密钥格式是否为 Base64
3. 消息是否被篡改

### Q: 发布后无法启动
**A:** 检查：
1. .NET 8 运行时是否安装
2. 使用 `--self-contained true` 发布
3. 查看事件日志获取详细错误
